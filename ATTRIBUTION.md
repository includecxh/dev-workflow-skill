- **Usage**: The bundled sub-skills in `bundled-skills/` are modified versions of skills originally from the Superpowers project. The original SKILL.md files have been customized to support dual-mode (merged/sequential) phase routing. The following files contain derivative code:
  - `bundled-skills/brainstorming/SKILL.md` — based on superpowers brainstorming skill

  - `bundled-skills/executing-plans/SKILL.md` — based on superpowers executing-plans skill
